SequalsK is a bidirectional transpiler between Swift and Kotlin.
For more details, see https://transpile.iem.thm.de/

This is a plugin for Android Studio that allows to access the
transpiler's functionality in a convenient way.

The first version of the plugin has been developed by Daniel Rhode,
see LICENSE-HISTORY.txt.

The current version of the plugin is maintained by Dominik Schultes,
see LICENSE.txt.
